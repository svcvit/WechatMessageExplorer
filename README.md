# WechatMessageExplorer
Mac 版微信聊天记录查看器

这是一个 Mac 版本的微信聊天记录查看实现，主要就是写着玩的，学习一下 macOS 的开发，所有数据查看都在本地，不会上传任何数据。

首先你需要获取自己微信本地数据库的密码，获取方式参考：[Mac 导出微信聊天记录](https://blog.vcvit.me/2024/08/02/mac-export-wechat-chat-records/)
